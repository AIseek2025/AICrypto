# skills/

平台正式 Skill 资产库。

## 家族分类

| 目录 | 家族 | 说明 |
|------|------|------|
| `trend/` | 趋势组 | 主升浪突破、回踩确认、高位离场等 |
| `short/` | 做空组 | 破位做空、反抽做空、事件做空等 |
| `correlation/` | 相关性组 | 板块补涨、龙头跟随、事件驱动等 |
| `risk/` | 风控组 | 降杠杆、缩仓、熔断、禁开仓等 |

## 纪律

- 每个 Skill 必须有 SkillSpec 定义
- 每个 Skill 必须绑定回测报告
- Skill 上线必须经过风控审批
